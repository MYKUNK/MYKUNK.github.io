坤坤安全引擎赋能安全，提供全方位安全保护

全面守护用户电脑安全的安全辅助软件

HOOK 驱动

安全软件的主要任务，就是要保护电脑不受病毒、木马这些恶意软件的侵害，除了能通过静态分析识别已知的威胁，还要守住计算机的安全防线，防止被恶意软件攻破。

智能填扑

有很部分文件因为在第一预览时没有加载到资源，又或者是JS动态调用出现了很多新的文件。不用担心在您浏览下载成果的时候。如果发现没有下载到的文件。会立即为您自动下载，这一切都不需要您的干预。软件会自动帮您完成，甚至您都不知道这回事。

主动防御驱动

光有一个HOOK驱动还不够，还得配套有主动防御驱动，负责完成具体的安全防御。

文件过滤驱动

通过HOOK驱动来进行监控，有时候并不能完全解决问题。\n因此，还要配套有文件过滤驱动，实现更底层的文件监控功能。

反病毒引擎

必不可少的就是反病毒引擎。安全软件最早的核心也就是这个东西，它的目的就是检测文件是不是恶意软件。

网络监控驱动

和文件过滤驱动类似，对于网络同样需要更底层的驱动程序，来监控所有的网络连接，监控所有的数据包，将网络通信情况完全了如指掌。

智能加速

电脑内存占用过高或临时文件达到1G时，自动清理内存和临时文件，减少电脑卡顿
